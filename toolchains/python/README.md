# Python

## Distribution

### Ubuntu

```bash
sudo apt install -y python3 python3-pip python3-venv python3-dev python-is-python3 pipx
pipx ensurepath
pipx install poetry
```

### Fedora

```bash
sudo dnf install -y python3 python3-pip python3-devel pipx 
pipx ensurepath
pipx install poetry
```
